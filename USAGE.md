# 婚礼筹备管家 - 使用说明

## 一、应用概述

婚礼筹备管家是一款专为婚礼策划师设计的跨平台应用，用于管理婚礼档期、客户信息和筹备流程。本应用支持Web端和移动端（iOS/Android），数据实时同步到云端。

## 二、主要功能

### 1. 管家档期管理

- **日历视图**: 直观查看全年婚礼档期
- **列表视图**: 按月份显示婚礼安排
- **统计视图**: 查看往年档期统计图表
- **添加档期**: 快速添加新的婚礼档期
- **本月档期**: 突出显示本月的婚礼安排
- **全年档期**: 查看全年的婚礼分布
- **本月收入**: 统计本月的收入情况

### 2. 客户详情管理

- **客户列表**: 查看所有客户信息
- **客户详情**: 查看和编辑客户的详细信息
- **状态管理**: 更新客户的筹备状态
- **搜索功能**: 快速搜索客户
- **客户操作**: 添加、删除和编辑客户信息

### 3. 核心筹备管理

- **筹备流程**: 管理婚礼筹备的各个环节
- **进度跟踪**: 记录筹备进度
- **任务分配**: 分配筹备任务
- **时间线管理**: 编辑和调整筹备时间线

### 4. 婚礼当天管理

- **当天流程**: 制定婚礼当天的详细流程
- **时间安排**: 安排各个环节的时间
- **人员分配**: 分配当天的工作人员
- **注意事项**: 记录重要的注意事项

## 三、使用指南

### 1. 首次使用

1. **访问应用**: 
   - Web端: 打开浏览器访问部署的Web地址
   - 移动端: 安装应用并打开

2. **初始化数据**: 
   - 首次使用时，应用会自动加载示例数据
   - 您可以根据需要修改或删除示例数据

3. **配置Firebase**: 
   - 确保您已经配置了Firebase数据库
   - 数据会自动同步到云端

### 2. 管理婚礼档期

1. **查看档期**: 
   - 点击导航栏中的"管家档期"选项
   - 切换日历视图、列表视图或统计视图

2. **添加档期**: 
   - 点击"添加婚礼档期"按钮
   - 填写新人信息、婚礼日期和场地
   - 点击保存

3. **编辑档期**: 
   - 在日历视图或列表视图中找到要编辑的档期
   - 点击档期项
   - 修改信息并保存

4. **删除档期**: 
   - 在列表视图中找到要删除的档期
   - 点击删除按钮
   - 确认删除

### 3. 管理客户信息

1. **查看客户列表**: 
   - 点击导航栏中的"客户详情"选项
   - 查看所有客户信息

2. **搜索客户**: 
   - 在搜索框中输入客户名称或其他关键词
   - 按回车键进行搜索

3. **查看客户详情**: 
   - 点击客户列表中的客户项
   - 查看客户的详细信息

4. **编辑客户信息**: 
   - 在客户详情页面，点击可编辑的字段
   - 修改信息
   - 点击保存按钮

5. **添加新客户**: 
   - 点击"添加客户"按钮
   - 填写客户信息
   - 点击保存

6. **删除客户**: 
   - 在客户详情页面，点击删除按钮
   - 确认删除

### 4. 管理核心筹备

1. **选择客户**: 
   - 首先在客户详情页面选择一个客户
   - 点击导航栏中的"核心筹备"选项

2. **查看筹备流程**: 
   - 查看该客户的筹备流程
   - 每个环节显示当前状态

3. **更新筹备进度**: 
   - 点击环节名称进入编辑页面
   - 更新环节的进度和状态
   - 点击保存

4. **添加筹备环节**: 
   - 点击"添加环节"按钮
   - 填写环节信息
   - 点击保存

5. **编辑筹备时间线**: 
   - 点击"编辑执行环节"按钮
   - 拖拽调整环节顺序
   - 点击保存

### 5. 管理婚礼当天

1. **选择客户**: 
   - 首先在客户详情页面选择一个客户
   - 点击导航栏中的"婚礼当天"选项

2. **查看当天流程**: 
   - 查看该客户婚礼当天的详细流程

3. **编辑当天流程**: 
   - 点击流程项进入编辑页面
   - 修改流程信息
   - 点击保存

4. **添加流程项**: 
   - 点击"添加流程项"按钮
   - 填写流程信息
   - 点击保存

## 四、数据同步

### 1. 自动同步

- 应用会自动将数据同步到Firebase云端
- 当您修改数据时，会自动保存到本地和云端
- 当您打开应用时，会自动从云端同步最新数据

### 2. 手动同步

- 您可以通过刷新页面手动触发数据同步
- 数据会从云端加载最新版本

### 3. 离线使用

- 应用支持离线使用
- 离线时，数据会保存在本地存储中
- 当网络恢复时，会自动同步到云端

## 五、导出功能

### 1. 导出数据

- 点击页面右上角的"导出"按钮
- 选择导出格式（Excel、PDF或图片）
- 按照提示完成导出

### 2. 分享功能

- 点击"分享"按钮
- 选择分享方式
- 分享应用链接或客户信息

## 七、常见问题

### 1. 数据不同步

- **解决方案**: 
  - 检查网络连接
  - 刷新页面
  - 检查Firebase配置
  - 检查数据库规则

### 2. 应用加载缓慢

- **解决方案**: 
  - 检查网络连接
  - 关闭其他应用
  - 清除浏览器缓存（Web端）
  - 清除应用缓存（移动端）

### 3. 无法添加新客户

- **解决方案**: 
  - 检查表单填写是否完整
  - 检查网络连接
  - 刷新页面重试

### 4. 页面显示异常

- **解决方案**: 
  - 刷新页面
  - 检查浏览器版本（Web端）
  - 更新应用版本（移动端）
  - 清除浏览器缓存（Web端）
  - 清除应用缓存（移动端）
